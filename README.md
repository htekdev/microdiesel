# Micro Diesel de Mérida

Sitio web oficial de **Micro Diesel de Mérida** — taller mecánico especializado en diésel en Mérida, Yucatán.

- **Domain:** [microdisel.mx](https://microdisel.mx)
- **Stack:** Astro 6 + Tailwind v4 + Vercel
- **Idioma:** Español (es-MX)

## Desarrollo

```bash
npm install
npm run dev
```

Servidor local en `http://localhost:4321`.

## Build

```bash
npm run build
```

Output estático en `dist/`.

## Branding

- Logo: `public/logo-mj-merida.jpg` (logo "MJ MÉRIDA" — paralelogramo verde, letras blancas)
- Color primario: verde MJ (`--color-mj-green`)
- Color secundario: azul marino oscuro (`--color-navy`)
- Tipografías: Montserrat (display) + Inter (body)
